MCP - Model Context Protocol

O protocolo padrão para conectar LLMs a fontes de dados e ferramentas. Saiba mais

MCP = Protocolo Universal

O Model Context Protocol é um padrão aberto que padroniza como LLMs se conectam a dados externos, definindo uma arquitetura cliente-servidor segura e bidirecional.

Pense no MCP como um "USB-C para IA", um conector universal que liga qualquer modelo a qualquer ferramenta.

MCP toolbox

A implementação prática do servidor MCP. Saiba mais

MCP toolbox = Implementação Prática

O MCP toolbox é a implementação do protocolo MCP no ADK, oferecendo auto-descoberta de ferramentas e gerenciamento de conexões.

Enquanto o MCP é o protocolo (como o HTTP), o MCP toolbox é a implementação (como um servidor web Apache).

ADK - Agent Development Kit

Framework do Google para desenvolvimento de agentes de IA.

ADK = Framework Completo

O Agent Development Kit é um framework open-source que oferece uma estrutura completa para criar agentes, com um WebServer integrado e suporte nativo ao MCP.

OpenAPI → tools.json

Conversão automática de especificações OpenAPI em ferramentas MCP

O Tradutor Automático

O projeto inclui uma ferramenta, `openapi-mcp-java`, que automatiza a conversão de arquivos de especificação OpenAPI (Swagger) em um arquivo `tools.json` compatível com o protocolo MCP. Isso elimina a necessidade de criar manualmente a definição de cada ferramenta.

🎯 O que é o BrainAPI?

O BrainAPI é um sistema que combina Inteligência Artificial conversacional com APIs REST, permitindo que usuários interajam com APIs utilizando linguagem natural. Ao invés de fazer chamadas HTTP manuais, você pode simplesmente perguntar e o sistema entende, identifica a API correta, executa a chamada e retorna a resposta em linguagem natural.

Arquitetura

Entenda como os componentes do BrainAPI se conectam.

🔄 Fluxo de Funcionamento

flowchart LR Usuario["👤 Usuário"] --> AgentADK["🧠 BrainAPI Agent"] subgraph "Geração de Ferramentas (Offline)" direction LR OpenAPI["📄 openAPI.json"] --> Gerador["⚙️ openapi2mcp"] Gerador --> Tool["📦 tools.json"] end subgraph "Execução (Online)" direction LR AgentADK --> LLM["🤖 LLM (Gemini)"] LLM --> MCPToolbox["🔧 MCP Toolbox"] Tool --> MCPToolbox MCPToolbox --> API["🔌 API Externa"] end
  1. Especificação: As APIs são documentadas em formato OpenAPI.
  2. Geração: A ferramenta `openapi2mcp-toolbox` converte os endpoints em ferramentas MCP (`tools.json`).
  3. Hospedagem: O MCP Toolbox (em um container Docker) serve as ferramentas via protocolo MCP.
  4. Processamento: O ADK Agent recebe a pergunta do usuário, usa o LLM para interpretá-la e chama a ferramenta apropriada através do MCP Toolbox.
  5. Resposta: O resultado da chamada da API é formatado em linguagem natural e devolvido ao usuário.

Instalação e Configuração

Passo a passo para colocar o BrainAPI para funcionar.

1️⃣ Clone o Repositório e Configure a API Key

git clone https://github.com/cesarschutz/BrainAPI.git cd BrainAPI export GOOGLE_API_KEY="sua-chave-aqui"

2️⃣ Gere as Ferramentas MCP

# Navegue para o gerador e compile cd openapi2mcp-toolbox mvn clean package # Gere o arquivo tools.json na raiz do projeto java -jar target/openapi-mcp-1.0-jar-with-dependencies.jar > ../tools.json cd ..

3️⃣ Inicie o MCP Toolbox com Docker

export VERSION=0.6.0 docker pull us-central1-docker.pkg.dev/database-toolbox/toolbox/toolbox:$VERSION docker run -d -p 8080:8080 -v $(pwd)/tools.json:/tools.json --name BrainAPI-mcp us-central1-docker.pkg.dev/database-toolbox/toolbox/toolbox:$VERSION --tools-file=/tools.json

4️⃣ Execute o ADK Agent

# Navegue para o agente cd BrainAPI-agent # Execute com o WebServer na porta 8081 mvn compile exec:java "-Dexec.args=--server.port=8081"

Após esses passos, a interface web estará acessível em http://localhost:8081.
